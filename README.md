# Professional Portfolio | Shambhavi Dixit

A minimalist personal portfolio built to showcase my foundational skills in Frontend Development and UI/UX design. This project focuses on high-impact visual storytelling and clean, semantic code.

## 🚀 Key Features
*   **Minimalist UI:** A "purpose-driven" dark-themed layout designed for clarity and impact.
*   **Responsive Design:** Optimized to ensure the portfolio remains functional and aesthetic across devices.
*   **Pure CSS Animations:** Showcases interactive elements like the Astro-Web Simulation using CSS Keyframe animations.

## 🛠️ Tech Stack
*   **HTML5:** Semantic structure for better accessibility and SEO.
*   **CSS3:** Custom styling including Flexbox, Grid, and Keyframe animations.
*   **Design Tools:** Styled with inspiration from Figma prototypes.
